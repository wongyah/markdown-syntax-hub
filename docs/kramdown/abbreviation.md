## 语法描述

在文中的任意位置为缩写添加一个定义（定义全称的语句），即可。

为缩写添加定义的语法如下：

{%- highlight markdown -%}
*[缩写]: 缩写的全称
{%- endhighlight -%}

## 示例

{%- highlight markdown -%}
哇，这就是我的 YYDS!

*[YYDS]: 永远滴神
{%- endhighlight -%}
